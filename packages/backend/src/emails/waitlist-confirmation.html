Hello, I'm Luca. I'm building KiwiClient because I want control of my email and my data - and I suspect you do too.

KiwiClient will be fast, open source, and shaped around the people who actually use it. There are good open-source email clients out there already; my hope is to build one that fits how you actually work.

I'll email you again when KiwiClient is ready to try. Until then, you're welcome to follow along, or contribute, on GitHub.

If you have any thoughts on what you'd like to see, or any questions, I'd love to hear them. Just reply to this email; it's only me building KiwiClient (so far!), so every reply lands directly with me.

A quick note on access when beta opens - KiwiClient will work straight away for any mail server that supports standard IMAP and SMTP, whether that's your own server, a small mail host, or any provider that gives you inbox credentials.

If you signed up with a Gmail address, full Gmail access requires Google's verification process, which costs tens of thousands of pounds a year and isn't sustainable for a non-commercial project like this one. So I'll be adding the first 100 Gmail signups to Google's tester list on a first-come basis when the beta opens. If you're beyond that cap, you'll be able to self-host your own KiwiClient instance (the code is AGPL-licensed, so anyone can run their own) or wait for Microsoft Outlook support, which is planned next.
